Автоматизированный расчётный комплекс

Приложение предназначено для автоматизации инженерных расчётов в среде Ansys Workbench с использованием журнальных файлов формата .wbjn.

Система позволяет изменять параметры расчётной модели, автоматически запускать расчёт и получать основные результаты моделирования без необходимости ручной настройки проекта в интерфейсе Ansys Workbench.

Основные возможности

  • Загрузка проектов Ansys Workbench
  • Работа с журналами .wbjn
  • Изменение параметров расчёта
  • Автоматический запуск моделирования
  • Получение результатов расчёта
  • Пошаговая визуализация результатов

Порядок работы

  1. Выберите файл проекта (*.wbpj)
  2. Загрузите файл журнала (*.wbjn)
  3. Введите параметры модели
  4. Запустите расчёт
  5. Просмотрите результаты моделирования
Примечание: приложение предназначено для автоматизации типовых задач статической прочности, уменьшения количества ручных операций и повышения удобства взаимодействия с системой Ansys Workbench.
