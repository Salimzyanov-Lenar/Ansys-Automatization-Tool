Документация пользователя

Данная система предназначена для автоматизации инженерных расчётов в среде Ansys Workbench с использованием журнальных файлов .wbjn.

Приложение позволяет изменять параметры модели, запускать расчёты, создавать эксперименты и получать результаты моделирования через единый веб-интерфейс.

1. Подготовка проекта Ansys Workbench

Перед началом работы необходимо создать и настроить расчётный проект в Ansys Workbench.

  1. Создайте расчётную модель в Ansys Workbench
  2. Настройте геометрию, нагрузки и граничные условия
  3. Параметризируйте необходимые величины расчёта
  4. Сохраните проект в формате .wbpj
  5. Запишите журнал действий в формате .wbjn

Рекомендуется параметризировать нагрузки, размеры модели и другие входные параметры, которые будут изменяться при выполнении расчётов.

2. Загрузка пользовательской задачи

После подготовки проекта необходимо загрузить журнальный файл в систему.

  1. Перейдите в раздел создания задачи
  2. Загрузите файл журнала .wbjn
  3. Выберите связанный проект .wbpj
  4. Сохраните задачу в системе

После сохранения задача станет доступна для запуска расчётов и изменения параметров.

3. Изменение параметров и запуск расчёта

Пользователь может изменять параметры модели без необходимости открытия Ansys Workbench.

  1. Откройте список доступных задач
  2. Нажмите кнопку редактирования параметров
  3. Введите необходимые значения
  4. Сохраните изменения
  5. Запустите расчёт задачи

Во время выполнения расчёта система автоматически запускает Ansys Workbench и выполняет расчёт с указанными параметрами.

4. Создание экспериментов

Система поддерживает проведение серии расчётов с различными параметрами модели.

  1. Создайте эксперимент для выбранной задачи
  2. Укажите набор параметров расчёта
  3. Запустите выполнение эксперимента
  4. Дождитесь завершения серии расчётов

Это позволяет выполнять анализ влияния параметров на результаты моделирования и ускоряет проведение инженерных исследований.

5. Получение результатов

После завершения расчёта доступны основные результаты моделирования.

  • Просмотр результатов расчёта
  • Выгрузка результатов в формате CSV
  • Построение графиков расчёта
  • Сохранение графиков
  • Анализ параметров эксперимента

Результаты могут использоваться для дальнейшего анализа, построения отчётов и обработки данных.

Примечание: настройка исполнителя расчётов, путей к Ansys Workbench и системных параметров выполняется через административную панель Django.
